# StudyTrack — AI Developer Learning OS Implementation Plan

> **Product Name**: StudyTrack  
> **Design**: Dark theme (#0A0A0A) with amber/orange (#D4A043) accents  
> **Backend**: Node.js (Express + TypeScript) with Prisma ORM

> **Deployment**: Render.com

## User Review Required

> [!IMPORTANT]
> **Scope Decision**: This is an enormous 26-module enterprise platform. Building everything at once is not realistic in a single session. I recommend a **phased approach** — delivering a fully working MVP first (Phase 1), then iterating.

> [!IMPORTANT]
> **Backend Simplification**: Your spec mentions Java Spring Boot + Python FastAPI + Node.js. However, you also stated **"Node.js for backend"**. I'll use **Node.js/Express** as the single backend, which simplifies deployment to Render significantly.

> [!WARNING]
> **AI Features**: Features like AI Tutor, Document Intelligence, and AI Learning Planner require an LLM API key (e.g., OpenAI, Google Gemini). You'll need to provide an API key. I'll build the integration layer, and you configure the key.

> [!WARNING]
> **Render Free Tier Limits**: Render's free tier has cold starts (~30s), 512MB RAM, and auto-sleep. The plan is designed to work within these limits, with Redis/background jobs added in later phases.

### 6. Render Deployment Configuration

#### [NEW] `render.yaml` (root) — Render Blueprint
```yaml
services:
  - type: web
    name: studytrack-api
    runtime: node
    buildCommand: cd backend && npm install && npx prisma generate && npx tsc
    startCommand: cd backend && node dist/server.js
    envVars:
      - key: DATABASE_URL
        fromDatabase:
          name: studytrack-db
          property: connectionString
      - key: JWT_SECRET
        generateValue: true
      - key: NODE_ENV
        value: production

  - type: web
    name: studytrack-web
    runtime: node
    buildCommand: cd frontend && npm install && npm run build
    startCommand: cd frontend && npm start
    envVars:
      - key: NEXT_PUBLIC_API_URL
        value: https://studytrack-api.onrender.com

databases:
  - name: studytrack-db
    plan: free
    databaseName: studytrack
```
