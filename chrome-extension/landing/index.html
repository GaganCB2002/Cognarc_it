Frequently Asked Questions

Is my data private?
100%. Everything stays in your browser's local database. Nothing is uploaded to any server.
How fast is document processing?
Most PDFs and text files process in under a second. Image OCR takes 1-5 seconds depending on size.
What file types are supported?
PDF, PNG, JPG, JPEG, GIF, BMP, WebP, TXT, MD, CSV, JSON, XML, HTML, and source code files.
Is there a limit on storage?
The extension uses your browser's IndexedDB, which typically allows hundreds of MBs.
Does it work offline?
Yes! Once installed, text extraction and document viewing work entirely offline.
